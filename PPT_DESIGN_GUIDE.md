# BidOrbit PowerPoint Design Guide

## Color Scheme

### Primary Colors
- **Primary Blue:** #2563EB (RGB: 37, 99, 235)
- **Dark Blue:** #1E3A8A (RGB: 30, 58, 138)
- **Light Blue:** #DBEAFE (RGB: 219, 234, 254)

### Secondary Colors
- **Success Green:** #10B981 (RGB: 16, 185, 129)
- **Warning Orange:** #F59E0B (RGB: 245, 158, 11)
- **Danger Red:** #EF4444 (RGB: 239, 68, 68)
- **Grey:** #6B7280 (RGB: 107, 114, 128)

### Background Colors
- **White:** #FFFFFF
- **Light Grey:** #F9FAFB
- **Dark Background:** #1F2937

## Typography

### Fonts
- **Headings:** Montserrat Bold or Poppins Bold
- **Body Text:** Open Sans or Roboto
- **Numbers:** Roboto Mono or Source Code Pro

### Font Sizes
- **Title Slide:** 48-60pt
- **Section Headers:** 36-44pt
- **Slide Titles:** 28-32pt
- **Body Text:** 18-24pt
- **Captions:** 14-16pt

## Slide Layouts

### Title Slide
- Full-screen gradient background (Blue to Dark Blue)
- Large centered title
- Subtitle below
- App mockup on right side
- Logo in corner

### Content Slides
- White or light grey background
- Blue header bar with slide title
- Content area with bullet points or images
- Footer with page number and logo

### Section Dividers
- Full-screen gradient
- Large section title
- Icon or illustration
- Minimal text

### Data Slides
- Charts and graphs
- Use brand colors
- Clear labels
- Source citations

## Visual Elements

### Icons
- Use consistent icon style (line icons or filled)
- Material Design icons recommended
- Size: 48-64px for main icons
- Color: Match brand colors

### Images
- High-quality screenshots of app
- Mobile mockups showing features
- Team photos (professional)
- Infographics for data

### Charts & Graphs
- Bar charts for comparisons
- Line charts for trends
- Pie charts for percentages
- Use brand colors consistently

## Slide-by-Slide Recommendations

### Slide 1 (Title)
- Gradient background
- App mockup (iPhone/Android)
- Animated logo entrance

### Slide 2-4 (Problem/Solution/Market)
- Icons for each point
- Statistics in large numbers
- Minimal text, maximum impact

### Slide 5-9 (Product Features)
- Screenshots of actual app
- Annotated with callouts
- Before/after comparisons
- Feature icons

### Slide 10-15 (Technical)
- Architecture diagrams
- Technology logos
- Code snippets (minimal)
- Performance graphs

### Slide 16-20 (Business)
- Financial charts
- Market size graphics
- Revenue model diagram
- Timeline/roadmap

### Slide 21-25 (Team/Traction)
- Team photos
- Testimonial quotes
- Metrics dashboard
- Growth charts

### Slide 26-30 (Vision/Ask)
- Inspiring imagery
- Clear call-to-action
- Contact information
- QR code for demo

## Animation Guidelines

### Entrance Animations
- Fade in for text
- Slide in from left for bullets
- Zoom in for images
- Duration: 0.5-0.75 seconds

### Emphasis Animations
- Pulse for important numbers
- Highlight for key points
- Grow/shrink for icons

### Exit Animations
- Fade out
- Slide out
- Duration: 0.3-0.5 seconds

### Transitions
- Fade between slides
- Push for section changes
- Duration: 0.5 seconds
- Don't overuse

## Best Practices

### Do's
✅ Use high-quality images
✅ Keep text minimal (6 words per line, 6 lines per slide)
✅ Use consistent branding
✅ Include page numbers
✅ Add speaker notes
✅ Test on projector
✅ Have backup PDF

### Don'ts
❌ Don't use too many fonts
❌ Don't overcrowd slides
❌ Don't use low-res images
❌ Don't read slides verbatim
❌ Don't use distracting animations
❌ Don't forget to proofread

## Presentation Tips

### Before Presentation
1. Practice timing (aim for 20-30 minutes)
2. Prepare for Q&A
3. Test equipment
4. Have demo ready
5. Print handouts

### During Presentation
1. Make eye contact
2. Speak clearly and slowly
3. Use presenter view
4. Engage audience
5. Handle questions confidently

### After Presentation
1. Share slides
2. Follow up with contacts
3. Send thank you notes
4. Gather feedback
5. Update materials
