## 🚀 How to Create Your PowerPoint

### Step 1: Open PowerPoint
- Open Microsoft PowerPoint
- Choose "Blank Presentation"
- Set slide size to 16:9 (Design → Slide Size)

### Step 2: Apply Theme
- Go to Design tab
- Choose a clean, modern theme (recommended: "Ion" or "Facet")
- Customize colors:
  - Primary: Blue (#2563EB)
  - Accent: Dark Blue (#1E3A8A)

### Step 3: Create Slides
Use the content from `BIDORBIT_PPT_GUIDE.md`:
- Each "SLIDE X" section = one PowerPoint slide
- Copy the content and format it nicely
- Add visuals (screenshots, icons, charts)

### Step 4: Add Visuals
**Required Screenshots:**
- Home screen with items
- Item details with countdown
- Bid placement dialog
- Notifications screen
- Profile screen
- Seller dashboard
- Add item screen

**Where to get them:**
- Take screenshots from the running app
- Use Android Studio emulator or real device
- Capture at 1080x1920 resolution

## 🎨 Quick Design Tips

### Colors to Use
- **Headers:** Blue gradient (#2563EB to #1E3A8A)
- **Text:** Dark grey (#1F2937)
- **Highlights:** Orange (#F59E0B) or Green (#10B981)
- **Background:** White or light grey (#F9FAFB)

### Fonts to Use
- **Titles:** Montserrat Bold, 32pt
- **Body:** Open Sans Regular, 20pt
- **Numbers:** Roboto, 48pt (for big stats)

### Layout Rules
- **Maximum 6 bullets per slide**
- **Maximum 6 words per bullet**
- **One main image per slide**
- **Consistent margins (1 inch)**

## 💡 Presentation Flow (20 minutes)

**Introduction (2 min)**
- Slides 1-2: Title, Problem

**Product (8 min)**
- Slides 3-9: Solution, Market, Features, Demo

**Business (5 min)**
- Slides 10-15: Model, Traction, Roadmap

**Ask (3 min)**
- Slides 16-18: Team, Financials, Investment

**Closing (2 min)**
- Slides 19-20: Vision, Thank You

**Q&A (10 min)**
- Be ready for questions
