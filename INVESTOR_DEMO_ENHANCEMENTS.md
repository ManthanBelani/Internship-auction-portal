## 📱 Demo Script for Investors

### Opening (30 seconds)
"Welcome to BidOrbit - a modern, real-time auction marketplace. Let me show you how seamless the experience is..."

**Action:** Open app → Auto-login → Home screen appears

### Discovery Phase (1 minute)
"Users can browse auctions by category, search for specific items, and see real-time countdowns..."

**Actions:**
1. Tap through categories (Art, Electronics, Watches)
2. Type in search bar → Show instant results
3. Scroll through items → Show smooth performance
4. Point out: Images, prices, countdown timers

### Engagement Phase (2 minutes)
"When users find something they like, they get detailed information and can bid instantly..."

**Actions:**
1. Tap an item → Item details screen
2. Swipe through images
3. Show countdown timer
4. Scroll to see description
5. Tap "Place Bid" → Show bid dialog
6. Enter amount → Submit
7. Show success message
8. Tap heart icon → Add to watchlist

### Tracking Phase (1 minute)
"Users can track all their activity in one place..."

**Actions:**
1. Tap "Bids" tab → Show 4 tabs (Winning, Outbid, Won, Ended)
2. Tap "Favourite" tab → Show watchlist
3. Tap bell icon → Show notifications
4. Tap "Profile" tab → Show stats and settings

### Seller Experience (1 minute)
"For sellers, we provide a complete dashboard to manage their auctions..."

**Actions:**
1. Logout → Login as seller
2. Show dashboard with stats
3. Tap "Add Item" → Show form
4. Upload images → Show preview
5. Fill details → Submit
6. Show inventory screen
