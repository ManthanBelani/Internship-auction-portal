## 📱 Demo Flow for Investors

### Buyer Journey (5 minutes)
1. Open app → Beautiful splash → Auto-login
2. Home → Browse categories → Smooth scrolling
3. Search → Find specific item
4. Item Details → See countdown, images, bids
5. Place Bid → Instant feedback
6. Watchlist → Add to favorites
7. Notifications → Real-time bid updates
8. Profile → View stats, settings
9. Won Items → Payment flow
10. Checkout → Complete purchase

### Seller Journey (3 minutes)
1. Switch to seller → Dashboard with stats
2. Add Item → Upload images, set price
3. Inventory → See all listings
4. Active Auctions → Monitor bids
5. Winners → Process sales
